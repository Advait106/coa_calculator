Arbitrary Precision Four-Function Calculator

This project demonstrates addition, subtraction, multiplication, and division with unlimited precision using a Flask backend implemented in Python. It performs digit-by-digit arithmetic rather than relying on built-in numeric types.

What is Arbitrary Precision?

Arbitrary precision arithmetic represents numbers as sequences of digits rather than fixed-size integers or floats. It only depends on available memory, not hardware word size, enabling accurate results on extremely large or small numbers.

Why is it Important?

  • Handles numbers larger than built-in types (hundreds or thousands of digits).
  • Removes rounding and overflow limits of native data types.
  • Crucial for cryptography, high-precision finance, scientific computing, and theoretical calculations.

Core Operations

  • Addition — Digit-by-digit with carry propagation.
  • Subtraction — Borrow handling and sign management.
  • Multiplication — Long multiplication (can be optimized using Karatsuba/FFT methods).
  • Division — Long division with configurable decimal precision.
